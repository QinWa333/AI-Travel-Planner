🚀 AI Travel Planner - 启动指南

==========================================
推荐方式：手动启动（最稳定）
==========================================

步骤 1: 启动后端
-----------------
打开 PowerShell 或 CMD，输入：

    conda activate ai-travel-planner
    cd backend
    python -m uvicorn app.main:app --reload

看到 "Application startup complete" 表示成功！
保持这个窗口运行！

步骤 2: 启动前端
-----------------
新开一个 PowerShell 或 CMD 窗口，输入：

    cd frontend
    npm run dev

看到 "Local: http://localhost:5173/" 表示成功！
保持这个窗口运行！

步骤 3: 访问应用
-----------------
打开浏览器访问：
- 前端: http://localhost:5173
- 后端 API 文档: http://localhost:8000/docs

==========================================
其他方式
==========================================

方式 2: 使用 PowerShell 脚本
----------------------------
在 PowerShell 中运行：

    .\start.ps1

如果提示无法运行，先执行：

    Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

方式 3: 使用 CMD 批处理
-----------------------
在 CMD（不是 PowerShell）中运行：

    start-conda.bat

注意：必须在 CMD 中运行！

==========================================
停止服务
==========================================

在各自的终端窗口按 Ctrl+C

==========================================
常见问题
==========================================

问题 1: conda 命令不存在
解决：使用 Anaconda Prompt 而不是普通 PowerShell

问题 2: 端口被占用
解决：
    netstat -ano | findstr :8000
    taskkill /PID <进程ID> /F

问题 3: 模块未找到
解决：
    conda activate ai-travel-planner
    cd backend
    pip install -r requirements.txt
